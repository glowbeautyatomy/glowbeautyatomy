# GlowBeautyAtomy - 글로벌 애터미 가입 가이드 사이트 설계

## 배경 / 목적

애터미(Atomy)는 전세계 27개 법인국 + 115개 가입 가능 지역에서 운영되는 글로벌 직접판매 기업이다. "Atomy sign up [country]", "애터미 가입 방법", "Atomy review" 같은 검색 의도를 가진 전세계 유저를 타겟으로, **신뢰감 있는 정보 제공 + 국가별 가입 가이드 + 제품 리뷰**를 통해 추천인 라인 가입을 유도하는 콘텐츠 사이트를 만든다.

이 사이트는 별도의 "퍼스널 브랜드(SNS 채널)"와는 분리된, **검색 트래픽 기반의 전환 도구**다. SNS 콘텐츠(인스타/유튜브)는 관계 기반 트래픽을 다루고, 이 사이트는 "이미 애터미를 알고 가입법을 찾는" 검색 기반 트래픽을 다룬다.

## 브랜드

- **이름**: GlowBeautyAtomy
- "Atomy"(검색 키워드 매칭) + "Glow Beauty"(K-뷰티 컨셉)를 결합
- GitHub 저장소/계정: `glowbeautyatomy`

## 비주얼 방향

- 그라데이션 애니메이션 히어로, 둥근 카드, 호버/스크롤 모션 효과 등 "힙한" K-뷰티 트렌드 톤
- 동시에 신뢰 요소(FAQ, 가입 가이드, 리뷰)를 명확한 섹션으로 노출하여, 검색 의도("사기 아니야?", "가입 방법") 트래픽의 전환율을 확보

## 기술 스택

- **프레임워크**: Next.js (App Router)
- **스타일링**: Tailwind CSS
- **애니메이션**: Framer Motion (히어로 그라데이션, 카드 등장/호버, 마퀴 배너)
- **배포**: Vercel (GitHub 연동, main 브랜치 푸시 시 자동 배포)
- **콘텐츠 관리**: 마크다운(.md/.mdx) 파일 기반 - 국가별 가이드, 리뷰 글 등은 콘텐츠 파일로 관리하여 코드 변경 없이 글 추가 가능

## 다국어(i18n) 구조

- v1: 한국어(`/ko/`), 영어(`/en/`)
- 처음부터 다국어 라우팅 구조를 잡아, 추후 베트남어/아랍어 등 추가 시 폴더만 추가하면 되도록 설계
- 언어 전환 스위처(헤더), 브라우저 언어 기반 자동 감지 + 수동 전환 가능
- 각 언어별 hreflang 메타데이터 자동 생성

## v1 페이지 구성

### 1. 홈 (`/`, `/ko`, `/en`)
- 그라데이션 애니메이션 히어로 (브랜드 소개 + 핵심 CTA)
- 신뢰 배지/마퀴 배너 (인기 국가 노출 - 호주, 알바니아, 브라질 등)
- 3개 핵심 섹션 카드: 국가별 가입 가이드 / 솔직 리뷰 / 신뢰 FAQ (호버 애니메이션)

### 2. 신뢰 FAQ (`/faq`)
- "애터미 사기 아니야?", "다단계인데 괜찮아?", "가입비 있어?" 등 검증/신뢰 단계 검색 의도에 대응
- 객관적이고 솔직한 톤 (장단점 모두 다룸)

### 3. 국가별 가입 가이드 (`/join/[country]`)
- v1: 한국(`/ko/join/korea`), 영어권 1~2개 (`/en/join/australia`, `/en/join/philippines` 우선 - 추후 키워드 검색량 확인 후 확정)
- 각 페이지 구성: 해당 국가에서의 가입 절차, 필요 정보(추천인 ID 등), 가입 시 받는 비금전적 혜택(정보/지원), CTA
- CTA 클릭 시 → 애터미 공식 글로벌몰(국가별 URL)로 리다이렉트, 추천인 코드를 쿼리 파라미터로 자동 첨부

### 4. 제품 리뷰/큐레이션 (`/reviews`, `/reviews/[product]`)
- 검색량 높은 제품(헴오힘 등) 위주의 리뷰/비교 콘텐츠
- 리뷰 내 가입/구매 CTA → 공식몰 링크 연동

## 추천인 코드 / 링크 관리

- 추천인 코드, 국가별 공식몰 URL은 `config/referral.ts` (또는 `.env`) 한 곳에서 관리
- 모든 CTA 컴포넌트는 이 설정을 참조 → 코드 변경 시 한 곳만 수정하면 전체 사이트 반영

## 범위 제외 (v1에서 다루지 않음)

- 커뮤니티(텔레그램 등) 연동 - 추후 단계
- 한국어/영어 외 추가 언어 콘텐츠 - 구조만 준비, 콘텐츠는 추후
- 결제/직접 판매 기능 - 모든 거래는 공식몰로 리다이렉트하여 처리 (자체 쇼핑몰 아님)

## 테스트/검증

- 로컬에서 `next dev`로 KO/EN 라우팅, hreflang, CTA 리다이렉트 동작 확인
- Vercel 프리뷰 배포로 모바일/데스크톱 반응형 확인
- Lighthouse로 SEO/성능 점수 확인 (애니메이션이 성능에 영향 주지 않는지 체크)
